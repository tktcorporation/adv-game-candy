# ゲームシステム全面改装 — 事例調査レポート + 設計案

## Context

**問題**: 学マスのスキルカードシステムを翻訳したが「カード選ぶ→数値UP→テキスト」の繰り返しで、選択に意味も面白さもない。
**要件**: ソシャゲとして完全に成立（デイリーループ・マネタイズ・周回やりこみ）。2-4時間のボリューム。
**ユーザー方針**: 学マス型が近い。前例のあるソシャゲモデルで。ちゃんとゲームにしたい。定食屋に縛らない。ひより主人公もOK。

## Part 2: 現行「灯」の診断

### 致命的に足りないもの

1. **クライマックスがない**: 学マスの試験、ウマ娘のレース、ときメモの告白に相当するものがない。毎日の積み重ねが「何のため」か不明
2. **掛け算がない**: バフ→大技の爆発感がない。+5が100回で+500。線形で退屈
3. **デッキ構築がない**: 毎日同じカードプールから3枚。戦略的思考がない
4. **プレイヤーの立場が曖昧**: 「常連客」は受動的。何をする人なのか不明

### 活かせるもの

- 3軸パラメータの概念
- 季節・天候システム
- 日替わりメニュー
- キャラクターの魅力（ひより、丸山）
- 「席の距離」の物語装置
- 既存のUI基盤・アニメーション

## Part 3: 設計案（v3 — ゲームループとストーリーの分離）

### 最重要設計原則: 2層構造

**前回の問題**: ストーリーとゲームループが1:1で結合（12週=12章）→ 拡張性ゼロ、継続運営不可能

**解決**: ブルアカ/あんスタと同じ「ゲーム部分」と「ストーリー報酬」の分離

```
【ゲームループ（毎日遊べる、ストーリーと独立）】
  ひよりの日常生活 → タスクをこなす → 「灯ポイント」を貯める
  ↓
【ストーリー報酬（灯ポイントを消費して解放）】
  灯に行く → ご飯を食べる → 晴や丸山との物語が進む
```

**なぜ分離が必要か**:
- シナリオと無関係にゲームだけ遊べる → 日常的なエンゲージメント
- シナリオ後もゲームが続く → 継続運営可能
- 新シナリオを追加するだけでコンテンツ拡張 → ゲームロジック変更不要
- 「ゲームを頑張った報酬として灯に行ける」→ 灯の特別感が増す

---

### 2層の詳細設計

### 【Layer 1】 ひよりの日常（ゲームループ）

毎日プレイするメイン部分。学マスの「レッスン」に相当。

**ひよりの1日** (1ターン = リアル1日分):
```
朝: 「今日はどう過ごす？」（行動カードから選択）
  → 図書館で仕事 / スーパーで買い出し / 読書 / 自炊に挑戦 / 散歩
昼: 行動の結果テキスト + パラメータ変化 + バフ獲得
夕: デイリーミッション確認 + 「灯ポイント」獲得
```

**学マスの4つの欠陥を修正**:

① **掛け算（バフシステム）**: 
```
「安心」: 共感系の効果×1.5（自炊成功、温かい食事で獲得）
「好奇心」: 理解系の効果×1.5（新しい本、発見で獲得）
「勇気」: 信頼系の効果×1.5（外出、新しい場所で獲得）
→ バフを積んでから大技 = 掛け算の爆発感
```

② **デッキ構築**:
```
初期デッキ: 8枚の基本行動カード（図書館/読書/買い物/自炊/散歩...）
進行で追加: 新しい行動カードを獲得（料理教室/古本屋巡り/公園ランチ...）
デッキ管理: カードの追加/削除/強化
→ 毎日の手札が「ひよりの生活の幅」を表す
```

③ **体力管理 =「心の余裕」**:
```
心の余裕: 行動するたびに消費。尽きると「引きこもりデー」（回復のみ）
高コスト行動（外出、新しいこと）は余裕を多く消費
低コスト行動（読書、自宅）は余裕を消費しにくい
→ 攻め（新体験）vs 守り（回復）のリソース管理
```

④ **ランダムイベント**:
```
天候: 雨の日は外出カードが弱体化 / 晴れの日は散歩カードが強化
季節: 春=花見カード出現、冬=鍋カード出現
偶然: 「帰り道で猫に会った」「古本屋でセール」等のランダム
→ 毎日違う判断が必要
```

**「灯ポイント」の獲得**:
- デイリーミッション達成: +3
- バフ付きカードプレイ: +1-2
- 特別イベント: +5
- ログインボーナス: +1
- **灯に行くのに必要**: 10ポイント（= 2-3日の日常で1回来店）

### 【Layer 2】 灯でご飯（ストーリー報酬）

灯ポイントを消費して「灯に行く」。ここがストーリーパート。

**来店の流れ**:
```
1. 灯ポイント10消費 → 灯に入る
2. 席選び（関係性に応じて選べる範囲が変わる）
3. 注文（日替わりメニューから選択）
4. ストーリーシーン（会話 + 選択肢）
5. 「ごちそうさま」→ 報酬（新カード、バフ、思い出）
```

**ストーリーとゲームの接点**:
- **来店するほど**: 晴との関係が進む → 新しいストーリーが解放
- **日常パートの結果が反映**: 
  - 自炊で料理スキルが上がっていると → 丸山との料理話が解放
  - 読書を多くしていると → ひよりがその本の話題を出せる
  - 勇気バフが高い状態で来店 → より大胆な選択肢が出現
- **灯で得たもの → 日常に還元**:
  - 新しい行動カード獲得（「晴くんがおすすめしてた本」→ 読書カード強化版）
  - パッシブ効果（「灯の常連」→ 毎日の安心+1）

### クライマックスの分離設計

**メインストーリー**: 灯の来店を重ねることで進行（22章）
- これは「報酬」なのでゲームループと独立
- 新章追加 = 新コンテンツ（ゲームロジック変更不要）

**週イベント（ゲーム側のクライマックス）**:
- 7日ごとに「週末チャレンジ」が発生
- その週の育成成果を試す5-8ターンの特別レッスン
- **学マスの試験に相当**
- ストーリーとは独立 → ストーリー完結後も毎週遊べる
- ランク判定（S/A/B/C）→ ボーナス灯ポイント + 限定報酬

**季節イベント（ライブサービスの核）**:
- 30日ごとの大型イベント
- 限定ストーリー + 限定カード + 限定報酬
- メインストーリーとは独立 → 永久に追加可能

### マネタイズ（分離構造を活かす）

| 商品 | 内容 | 層 |
|---|---|---|
| 行動カード(ガチャ) | デッキに追加する特別な日常カード | Layer 1 |
| サポートカード(ガチャ) | パッシブ効果。学マスと同じ構造 | Layer 1 |
| 灯チケット | 灯ポイントを即時獲得（ストーリーを早く読みたい人向け） | Layer 2 |
| 月額パス | デイリー灯ポイント+2 + 限定テキスト | 両方 |
| 衣装 | ひよりの着せ替え（日常・灯両方に反映） | 両方 |
| 季節パス | 季節イベントのフルアクセス + 限定報酬 | イベント |

### 「灯」の世界観での翻訳（v3）

| 学マス | 灯（改善版） |
|---|---|
| レッスン(Vo/Da/Vi) | ひよりの日常（読書/買い物/自炊/散歩）→ 理解/信頼/共感 |
| 好調/集中/やる気 | 安心/好奇心/勇気 |
| 体力 | 心の余裕 |
| SPレッスン | 特別な日（天気/季節/偶然のイベント） |
| 中間試験 | 週末チャレンジ（ゲーム側クライマックス） |
| 最終試験 | 季節イベントのフィナーレ |
| コミュ/True End | 灯でのストーリーシーン（Layer 2報酬） |
| デッキ圧縮 | 気持ちの整理（不要なカードを捨てる） |
| プロデュース1回 | 1週間サイクル（日常パート） |
| **新: ストーリー** | **灯ポイントで解放（ゲームと独立）** |

### 拡張性のまとめ

```
ゲーム完結時: メインストーリー22章完結
  → 日常パート（Layer 1）は永久に遊べる
  → 週末チャレンジは毎週発生
  → 季節イベントで新ストーリーを追加可能
  → 新しい行動カードを追加可能（ガチャ更新）
  → 後日談/IF/別キャラ視点をストーリーとして追加可能
```

## Part 4: 実装計画（概要）

### Phase 1: データ型刷新
- `types.mbt`: 週サイクル、気持ちバフ、クライマックスイベント型の追加
- `state.mbt`: デッキ管理、バフスタック、週進行の追加

### Phase 2: コアメカニクス
- デッキ構築ロジック（カード追加/削除/強化）
- バフシステム（安心/好奇心/勇気のスタックと掛け算）
- クライマックス判定（金曜イベントのスコア計算）
- 心の余裕（体力）管理

### Phase 3: 週サイクル管理
- 月〜木の日常ターン進行
- 金曜のクライマックスイベント進行
- 週リセット（デッキリセット、新しい週の開始）

### Phase 4: UI
- 日常ターンUI（カード選択 + ひよりの心の声テキスト）
- クライマックスUI（特別演出 + 判定結果）
- 週進行表示（カレンダー風）
- デッキ管理画面

### Phase 5: シナリオ・テキスト
- 12週分のクライマックスシナリオ（金曜イベント）
- カード使用時のひよりのモノローグ
- バフ発動時の演出テキスト
