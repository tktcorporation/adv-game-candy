# 灯（あかり）

定食屋「灯」を舞台にした、ノベル×ソーシャルゲーム。

毎日通って、日替わりを食べて、隣の席のあの人と少しずつ近づいていく——そんな物語。

**[プレイする](https://candy-adv.netlify.app)**

## ゲーム概要

路地裏の小さな定食屋「灯」。カウンター8席、テーブル2つ。日替わり定食が一種類だけ。

主人公の高瀬晴は、引っ越してきたばかりの町でこの店を見つけた。ほぼ毎日通うようになった頃、同じ時間帯に来るもう一人の常連——椎名ひよりの存在に気づく。

カウンターの端と端。名前も知らない。目も合わない。

でも、毎日同じ空間にいるだけで、何かが少しずつ変わっていく。

### ゲームシステム

- **AP行動選択制**: 毎日5APの中で「食べる」「観察する」「丸山と話す」を選ぶ
- **親密度システム**: 行動の積み重ねでひよりとの距離が縮まる（★1〜★5）
- **ストーリー解放**: 親密度が一定に達すると本編シナリオが開放される
- **デイリーミッション**: 毎日の小さな目標が親密度ボーナスに
- **味の記憶**: 食べた日替わりメニューや発見がコレクションに

## 開発

### 前提条件

- [MoonBit](https://www.moonbitlang.com/) ツールチェーン
- Node.js

### ビルド & プレビュー

```bash
# 依存の取得
moon update && npm install

# ビルド（JS バンドル生成）
moon build --target js --release && npx rolldown -c rolldown.config.mjs

# ローカルプレビュー
npx serve dist
```

### 開発コマンド

```bash
moon check                    # 型チェック（高速）
npm run lint:scenario         # シナリオ lint
```

### プロジェクト構成

```
src/
  data/       シナリオ（10章）、キャラクター設定、日替わりメニュー
  game/       ゲームロジック（AP管理、親密度、行動処理）
  ui/         UI コンポーネント
  lint/       シナリオリンター
  main/       エントリーポイント
```
