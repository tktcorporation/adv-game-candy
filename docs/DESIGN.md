# 廃墟カフェ復興記 - ゲームデザインドキュメント

## コンセプト
「場所は人が集まることで生きる」

廃墟となったカフェ「月灯り」を復興させる、ストーリー駆動のカフェ経営ゲーム。
ノベルADVのナラティブ、リアルタイムスタミナシステム、ソーシャルゲーム的メカニクスを組み合わせた作品。

## ゲームフェーズ

```
GamePhase::Story
  ├─ ストーリーテキスト表示（ノベルADVモード）
  ├─ シーンインデックス追跡
  ├─ 行ごとの進行
  └─ 完了 → GamePhase::Business

GamePhase::Business
  ├─ デイリースタミナ表示
  ├─ メニュー選択
  ├─ 「営業開始」ボタン
  ├─ スタミナ消費チェック
  └─ 成功 → GamePhase::DayResult

GamePhase::DayResult
  ├─ 客来訪ログ表示
  ├─ 売上/原価サマリ
  ├─ 「次の日へ」ボタン
  └─ → 新日、Businessフェーズへリセット
```

## コアシステム

- 最大: 100ポイント

- 全回復: 約10時間
